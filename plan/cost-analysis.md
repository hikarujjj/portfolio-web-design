## 顧客生涯価値（LTV）分析

### LTV計算

```
平均月額: $19（Basic） or $49（Pro）
平均顧客寿命: 1 / チャーン率

LTV（Basic, 15%チャーン） = $19 / 0.15 = $127
LTV（Pro, 15%チャーン） = $49 / 0.15 = $327
```

### 顧客獲得コスト（CAC）

**オーガニック獲得**:
- コスト: ほぼゼロ（時間のみ）
- CAC: $0-5

**有料広告（将来）**:
- Google広告 CPA: $20-30
- SNS広告 CPA: $15-25
- CAC: $15-30

### LTV/CAC比率

```
オーガニック:
LTV/CAC = $127 / $5 = 25.4（優秀）

有料広告:
LTV/CAC = $127 / $25 = 5.08（良好）
```

**目標**: LTV/CAC > 3.0（健全なビジネス）
