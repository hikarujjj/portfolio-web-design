# BONBON ATELIER - Candy Shop Landing Page

## デザインコンセプト

**Aesthetic Direction**: "Sweet Sophistication"
60年代キャンディーショップのノスタルジアと現代的なミニマリズムの融合。パステルカラーの柔らかさとガラスモーフィズム効果により、ポップでありながら洗練された高級感を演出。

**Target Audience**: 大人向けの上質なキャンディーを求める層。ギフト需要、自分へのご褒美需要。

**Brand Tone**: カラフル、ポップ、でも落ち着いた上品さ。遊び心と洗練の両立。

## Frontend Aesthetics Guidelines

### カラーパレット

```css
--color-cream: #FFF8F0;      /* 背景 - 温かみのあるクリーム */
--color-pink: #FFB8D1;        /* アクセント1 - パステルピンク */
--color-mint: #B8E6D5;        /* アクセント2 - ミントグリーン */
--color-lavender: #D4B5F0;    /* アクセント3 - ラベンダー */
--color-peach: #FFD4B8;       /* アクセント4 - ピーチ */
--color-blue: #B8D4FF;        /* アクセント5 - ライトブルー */
--color-yellow: #FFF4B8;      /* アクセント6 - クリームイエロー */
--color-plum: #4a3545;        /* テキストメイン - 深いプラム */
--color-plum-light: #6b5060;  /* テキストサブ */
```

**カラー使用方針**:
- 背景は常にクリーム系で統一し、清潔感と温かみを
- 各セクションで異なるパステルアクセントを使用（単調さを避ける）
- グラデーション多用: 2-3色のパステルカラーを組み合わせる
- テキストは深いプラム色で可読性を確保

### タイポグラフィ

**Display Font**: "Playfair Display" (serif)
- 用途: ロゴ、見出し、セクションタイトル
- 特徴: エレガントで遊び心のあるセリフ体
- ウェイト: 400 (Regular), 600 (SemiBold), 700 (Bold)

**Body Font**: "Plus Jakarta Sans" (sans-serif)
- 用途: 本文、ナビゲーション、ボタン
- 特徴: モダンで読みやすいサンセリフ体
- ウェイト: 300 (Light), 400 (Regular), 500 (Medium), 600 (SemiBold)

**文字組み**:
- letter-spacing: 0.05em～0.15em（高級感を演出）
- 見出しは大胆に大きく（clamp使用でレスポンシブ対応）
- 行間は1.6～1.9で余裕を持たせる

### アニメーション・モーション

**ページロード時**:
- `fadeInUp`: 下から上へフェードイン（ヒーローセクション）
- スタガーアニメーション: `animation-delay`で順次表示
- カテゴリーカード、ギャラリーアイテムはスクロールで順次表示

**インタラクション**:
- ホバー: `translateY(-15px) scale(1.02)` - ふんわりとした浮遊感
- カード影の変化: `box-shadow`の濃さを変える
- トランジション: 0.3s～0.5s ease

**背景アニメーション**:
- `floating-candy`: 20秒の無限ループでゆっくり浮遊
- 複数の円形グラデーションが異なる遅延で動く

**カスタムカーソル**:
- デフォルト: 二重円（外側ピンク、内側ラベンダー）
- ホバー: サイズ拡大

### ビジュアル要素

**写真**:
- Unsplash使用: キャンディー、スイーツ、カラフルなお菓子の高品質写真
- 推奨サイズ: w=800～1200, q=80
- aspect-ratio使用で一貫性のあるレイアウト

**形状**:
- border-radius: 20px～30px（丸みを帯びた優しい印象）
- カードは完全な角丸で柔らかさを強調

**エフェクト**:
- グラスモーフィズム: `backdrop-filter: blur(10px)` (ヘッダー)
- グラデーションオーバーレイ: ホバー時に商品画像の上に表示
- 浮遊する背景装飾: radial-gradientの円形要素

### レイアウトパターン

**グリッドシステム**:
- カテゴリー: `grid-template-columns: repeat(auto-fit, minmax(320px, 1fr))`
- ギャラリー: `grid-template-columns: repeat(auto-fit, minmax(280px, 1fr))`
- 2カラムレイアウト: ストーリーセクション、店舗情報

**スペーシング**:
- セクション間: 120px（デスクトップ）、80px（モバイル）
- カード間: 40px（デスクトップ）、30px（モバイル）
- パディング: 5%の左右マージン

## セクション構成

### 1. Hero Section
- フルスクリーン（100vh）
- 中央配置
- グラデーションテキスト: ピンク→ラベンダー→ミント
- スクロールインジケーター付き

### 2. Categories Section
- 4つのカテゴリーカード（フルーツ、チョコレート、ミント、季節限定）
- 各カードに異なるグラデーションラベル
- ホバーで浮き上がり効果

### 3. Story Section
- 2カラムレイアウト（画像 + テキスト）
- ハイライトボックス: 創業者の言葉を強調

### 4. Gallery Section
- 6枚のギャラリー（3x2グリッド）
- ホバーでグラデーションオーバーレイ + テキスト表示
- 画像拡大効果

### 5. Store Info Section
- 営業時間とアクセス情報を2カラムカードで表示
- プレースホルダーマップ（実際のマップは後で追加可能）

### 6. Footer
- ダークプラム背景
- ロゴ、ナビゲーションリンク、SNSアイコン
- シンプルで洗練されたレイアウト

## 拡張時の注意点

### 新しいセクション追加時
- パステルカラーパレットから2-3色選んでグラデーション作成
- 背景は基本クリーム、アクセントセクションのみ薄いパステルグラデーション
- アニメーションは控えめに（ふんわり、ゆっくり）

### 新しいカテゴリー追加時
- `.category-card:nth-child(n) .category-label`でグラデーション指定
- 未使用のパステルカラーの組み合わせを使う
- 画像はUnsplashから高品質なものを選定

### レスポンシブ対応
- 768px以下でモバイルレイアウトに切り替え
- グリッドは1カラムに
- フォントサイズはclamp()で自動調整
- パディングとマージンを縮小

### パフォーマンス
- 画像はUnsplash URLのw=パラメータで最適化
- アニメーションはCSS onlyで実装（GPU加速）
- Intersection Observerでスクロールアニメーションを遅延実行

## 技術仕様

- **フレームワーク**: なし（Vanilla HTML/CSS/JS）
- **外部依存**: Google Fonts (Playfair Display, Plus Jakarta Sans)
- **画像**: Unsplash API
- **ブラウザ対応**: モダンブラウザ（Chrome, Firefox, Safari, Edge最新版）
- **レスポンシブ**: モバイルファースト、768pxブレークポイント
