# LUMIÈRE - 美容院ランディングページ

このドキュメントは、LUMIÈREランディングページのデザインガイドラインと制作意図を記録しています。関連ページや更新を行う際は、このガイドラインに従って一貫性を保ってください。

## デザインコンセプト

**トーン:** 洗練されたミニマリズム、エレガント、高級感、清潔感

美容院という業態の特性を活かし、清潔感と高級感を両立させたデザイン。白を基調とした明るく開放的な雰囲気の中に、ゴールドのアクセントで上品さと特別感を演出。訪問者に「ここに行きたい」と思わせる洗練された空間を表現。

## Frontend Aesthetics Guidelines

### トーン・方向性
- **洗練されたミニマリズム**: 過度な装飾を避け、余白と繊細な影で品質を表現
- **エレガント**: 流麗なセリフ体タイポグラフィで高級感を演出
- **清潔感**: 白を基調とした明るく開放的なデザイン
- **高級感**: ゴールドアクセントと繊細なアニメーションで特別感を創出

### カラースキーム

```css
--color-primary: #ffffff;      /* メインカラー（白） */
--color-secondary: #f8f6f3;    /* セカンダリ（ベージュ） */
--color-accent: #c9a969;       /* アクセント（ゴールド） */
--color-text: #2d2d2d;         /* テキスト（ダークグレー） */
--color-text-light: #6b6b6b;   /* サブテキスト（ライトグレー） */
```

**カラー戦略:**
- **白(#ffffff)**: 清潔感、開放感、プロフェッショナリズムの象徴
- **ベージュ(#f8f6f3)**: 柔らかさと温かみを加える背景色
- **ゴールド(#c9a969)**: 高級感と特別感を演出するアクセント
- **控えめな配色**: 全体的に彩度を抑え、上品で落ち着いた雰囲気に

### タイポグラフィ

```css
--font-display: 'Cormorant Garamond', serif;  /* 見出し */
--font-body: 'Montserrat', sans-serif;        /* 本文 */
```

**フォント選定理由:**
- **Cormorant Garamond**: 優雅で古典的なセリフ体。美容院の洗練さとエレガンスを表現
- **Montserrat**: モダンで読みやすいサンセリフ体。情報の明確な伝達

**タイポグラフィパターン:**
- 見出し: `letter-spacing: 0.1em～0.3em`（高級感のため）
- 本文: `font-weight: 300`（軽やかさを保つ）
- サイズ階層: `clamp()`を使用したレスポンシブスケーリング

### モーション・アニメーション

**アニメーション方針:**
- **スムーズで上品**: 急激な動きを避け、優雅な遷移を実現
- **段階的表示**: `animation-delay`で要素を順番に表示し、視線誘導
- **ホバーエフェクト**: `translateY(-10px)`で浮き上がる、繊細な影の変化

**実装パターン:**
```css
/* フェードインアップ（汎用） */
@keyframes fadeInUp {
    from { opacity: 0; transform: translateY(30px); }
    to { opacity: 1; transform: translateY(0); }
}

/* スライドダウン（ナビゲーション） */
@keyframes slideDown {
    from { transform: translateY(-100%); opacity: 0; }
    to { transform: translateY(0); opacity: 1; }
}

/* ズームイン（背景画像） */
@keyframes zoomIn {
    from { transform: scale(1.1); }
    to { transform: scale(1); }
}
```

### 画像・ビジュアル

**画像選定基準:**
- **ソース**: Unsplash（高品質、商用利用可）
- **テーマ**: 美容院インテリア、ヘアスタイル、ヘアケア
- **品質**: `?w=800&q=80`～`?w=1600&q=80`
- **雰囲気**: 明るく清潔感のある写真、プロフェッショナルな仕上がり

**ビジュアル戦略:**
- エモーショナルな要素として使用（絵文字❌、プロの写真⭕）
- オーバーレイで文字の可読性を確保
- ホバー時にズーム＋明度変更で没入感を演出

### 空間構成・レイアウト

**レイアウト原則:**
- **十分な余白**: セクション間 `padding: 8rem 5%`
- **中央揃え**: 最大幅 `max-width: 1200px`で読みやすさを確保
- **グリッドシステム**: `grid-template-columns: repeat(auto-fit, minmax(300px, 1fr))`でレスポンシブ対応
- **アスペクト比**: ギャラリー画像は `aspect-ratio: 4/5`で統一感

**背景・視覚的詳細:**
- **繊細な影**: `box-shadow: 0 5px 20px rgba(0,0,0,0.03)`で奥行きを表現
- **グラデーションオーバーレイ**: ヒーローセクションで画像と文字の調和
- **ホバー時の影強化**: `box-shadow: 0 15px 40px rgba(0,0,0,0.08)`でインタラクション

## セクション構成

### 1. ナビゲーション
- **固定ヘッダー**: `position: fixed`でスクロール時も表示
- **半透明背景**: `backdrop-filter: blur(10px)`で背景とブレンド
- **ロゴ**: レターステーシング `0.3em`で洗練された印象
- **ナビリンク**: ホバー時にアンダーラインアニメーション

### 2. ヒーローセクション
- **フルスクリーン**: `height: 100vh`でインパクトを最大化
- **背景画像**: 美容院インテリアのプロフェッショナル写真
- **オーバーレイ**: 白のグラデーションで文字の可読性確保
- **CTAボタン**: ゴールド背景、リップルエフェクト付き

### 3. サービスセクション
- **3列グリッド**: カット、カラー、トリートメント
- **カード型レイアウト**: 白背景、繊細な影で浮かせる
- **アイコン**: テキストベースのシンプルなアイコン（絵文字不使用）
- **価格表示**: Cormorant Garamondでエレガントに

### 4. ギャラリーセクション
- **3列グリッド**: ヘアスタイルとインテリアの写真
- **オーバーレイ**: ホバー時にゴールドのオーバーレイ表示
- **段階的表示**: `animation-delay`で順番にフェードイン

### 5. アクセス・営業時間
- **2列レイアウト**: 連絡先と営業時間を並列表示
- **情報の階層化**: ラベル（小さく）と内容（大きく）で視認性向上
- **営業時間テーブル**: 罫線で区切り、読みやすく整理

### 6. フッター
- **シンプル**: ロゴとコピーライトのみ
- **ダーク背景**: `background: var(--color-text)`でコントラストを作る

## 今後の拡張・更新時の注意点

### デザインの一貫性を保つために

1. **カラー**: 必ず定義済みのCSS変数を使用。新色追加は慎重に
2. **フォント**: Cormorant Garamond × Montserratの組み合わせを維持
3. **アニメーション**: 既存のキーフレームを再利用。新規追加は同じトーンで
4. **余白**: セクション間は `8rem`、要素間は `2rem～3rem`を基準に
5. **画像**: Unsplashから同じ雰囲気の写真を選定

### 追加セクションの例

新しいセクション（例: スタッフ紹介、お客様の声など）を追加する場合:
- 既存のセクション構造（`.section-title` + `.section-subtitle` + グリッド）を踏襲
- カード型レイアウトで統一感を保つ
- ホバーエフェクトは `translateY(-10px)`パターンを使用
- アニメーション遅延は `0.1s`刻みで設定

### 避けるべき要素

- ❌ 絵文字（🌟✨など）- プロフェッショナルさを損なう
- ❌ 強い彩度の色（ネオンカラー等）- 清潔感を損なう
- ❌ 過度なアニメーション - エレガンスを損なう
- ❌ ごちゃごちゃした背景パターン - ミニマリズムに反する
